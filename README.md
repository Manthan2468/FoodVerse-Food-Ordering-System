# 🍔 FoodVerse - Food Ordering System

FoodVerse is a web-based food ordering application developed using Python and Django. The system is designed to simplify the online food ordering process by allowing users to browse food items, place orders, and manage their purchases through a user-friendly interface. The project demonstrates the use of Django's architecture, database management, user authentication, and full-stack web development concepts.

## 🚀 Features

* User Registration and Login
* Browse Food Menu
* Food Category Management
* Search Food Items
* Add to Cart Functionality
* Order Placement and Management
* User Authentication and Authorization
* Responsive User Interface
* Database-Driven Application
* Admin Dashboard for Management

## ⚙️ Installation Guide

### Clone Repository

```bash
git clone https://github.com/your-username/FoodVerse-Food-Ordering-System.git
cd FoodVerse-Food-Ordering-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Development Server

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
